Vantage UI Stack

Stack Overview

Icons

Tabler Icons

5,000+ のオープンソースアイコン。シャープでモダンなデザイン。

  • 5,000+ アイコン
  • Solid.js 対応
  • SVG / stroke-based
  • tree-shakeable
Primitives

Kobalte

Solid.js用のアクセシブルなUIプリミティブ。WAI-ARIA準拠。

  • 35+ コンポーネント
  • WAI-ARIA 準拠
  • スタイル非依存
  • カスタマイズ自由
Extra

corvu

Kobalteを補完する専門コンポーネント。DX重視。

  • Drawer (モバイル対応)
  • Resizable パネル
  • Calendar / DatePicker
  • Kobalte互換API
