# 三言語対応ルール

このファイルは、本プロジェクトのドキュメント作成における言語対応ルールを定義します。

## 対象言語

- 日本語 (Japanese) - 翻訳元
- 英語 (English)
- 韓国語 (Korean)

## 翻訳ガイドライン

### 技術用語

- 一般的な技術用語は各言語の標準的な表現を使用
- Apple固有の用語は公式ドキュメントに準拠

### トーン

- 丁寧語を使用
- メッセージのやりとりは、気さくな感じで。
- 笑顔とユーモアを忘れない。ただタイミングを間違えないでね。

## 実装例

```markdown
# Vantage

## 概要

VantageはApple Vision Pro向けの没入型アプリケーションです。

## 機能

- カスタムMetalレンダリング
- ARKit統合
- RealityKit対応
```

## メンテナンス

- 新機能追加時は必ず三言語でドキュメントを更新
- 翻訳の一貫性を保つため、用語集を維持
