# Contributor Covenant Code of Conduct

## 私たちの誓い

私たちメンバー、コントリビューター、リーダーは、年齢、体格、目に見える / 見えない障害、民族性、性別、性自認 / 性表現、経験レベル、教育、社会経済的地位、国籍、容姿、人種、カースト、肌の色、宗教、性的指向に関わらず、私たちのコミュニティへの参加をハラスメントのない体験にすることを誓います。

オープンで歓迎的、多様で包摂的、健全なコミュニティを育む形で行動し、交流することを誓います。

## 私たちの基準

私たちのコミュニティに対してポジティブな環境に貢献する行動の例:

- 他者への共感と親切さを示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを与え、優雅に受け入れること
- 自分の過ちの影響を受けた人々に責任を持って謝罪し、その経験から学ぶこと
- 私たち個人だけでなく、コミュニティ全体にとって最善のことに焦点を当てること

容認できない行動の例:

- 性的な言葉やイメージの使用、あらゆる種類の性的注目や接近
- トロール行為、侮辱的または軽蔑的なコメント、個人攻撃や政治攻撃
- 公的または私的なハラスメント
- 明示的な許可なく、住所や電子メールアドレスなど他者の個人情報を公開すること
- 専門的な場で合理的に不適切と考えられるその他の行為

## 執行の責任

コミュニティリーダーは、私たちの受け入れられる行動の基準を明確にし、執行する責任を持ち、不適切、脅迫的、攻撃的、または有害と判断したあらゆる行動に対して、適切で公正な是正措置を取ります。

## 適用範囲

この行動規範は、すべてのコミュニティの場で適用され、また個人が公の場でコミュニティを公式に代表する場合にも適用されます。

## 執行

虐待的、ハラスメント、その他容認できない行動は、執行責任者である **mito@chronista.club** に報告される場合があります。すべての苦情は迅速かつ公正にレビューされ、調査されます。

すべてのコミュニティリーダーは、いかなる事件の報告者のプライバシーとセキュリティも尊重する義務があります。
